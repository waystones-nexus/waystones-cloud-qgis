about=Waystones Cloud turns any QGIS vector layer into live, shareable web services in seconds.

  Select a layer, enter a dataset slug, choose your services, and click Publish:
  - OGC API Features (OAPIF) — a queryable REST API for your vector data; the modern successor to WFS
  - Vector tiles (MVT / PMTiles) — fast, zoomable tiles ready for web maps and GIS clients
  - STAC — standardised spatial metadata for data discovery and cataloguing

  Works with any vector source QGIS can open: shapefiles, GeoJSON, GeoPackage, PostGIS, KML, GML, and more. The plugin converts to GeoPackage automatically — no manual export needed.

  Requires a Waystones Cloud account and API key. Sign up at https://waystones.cloud.
